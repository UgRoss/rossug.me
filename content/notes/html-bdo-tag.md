The `<bdo>` (Bidirectional Text Override) element overrides text directionality.

## Usage

```html
<bdo dir="rtl">Hello</bdo> World!
<!-- Displays: olleH World! -->
```

## Direction attribute

| Value       | Effect        |
| ----------- | ------------- |
| `dir="rtl"` | Right to left |
| `dir="ltr"` | Left to right |
