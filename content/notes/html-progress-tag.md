pubDate: '2020-12-30'
category: 'HTML'
---

Represents task completion status as a visual progress bar.

## Attributes

| Attribute | Description                 | Default |
| --------- | --------------------------- | ------- |
| `max`     | Total work needed           | 1.0     |
| `value`   | Current progress (0 to max) | —       |

## Example

```html
<progress value="20" max="100"></progress>
<!-- Shows 20% completion -->
```
