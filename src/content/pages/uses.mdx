A minimalist overview of the tools, software, and hardware I use daily to build things and stay productive. This setup is constantly evolving.

![My minimalist workspace setup](./_assets/workspace_placeholder.png)
_My current workspace setup (placeholder)_

## Hardware

- **Computer:** 14" MacBook Pro (M2 Max) - My daily driver for everything.
- **Monitor:** 27" 4K Display - For that extra screen real estate when at my desk.
- **Keyboard:** Custom Mechanical Keyboard - Linear switches for a smooth typing experience.
- **Mouse:** Wireless Ergonomic Mouse - Essential for long coding sessions.
- **Audio:** Studio Monitor Headphones - Keeping things accurate and focused.

## Software

- **Editor:** Visual Studio Code - With a strictly minimalist theme and essential extensions.
- **Terminal:** iTerm2 + Zsh - A highly customized shell for maximum CLI efficiency.
- **Browser:** Arc - Rethinking how a browser should work in 2024.
- **Notes:** Obsidian - My second brain for everything from project ideas to random thoughts.
- **Design:** Figma - For quick UI mockups and design iterations.

## Productivity

- **Task Management:** Linear - Keeping projects on track with speed and precision.
- **Calendar:** Cron - A fast and beautiful way to manage my schedule.
- **Communication:** Slack & Discord - Where most of the collaboration happens.
